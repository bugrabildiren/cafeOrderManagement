#Project Proposal

## Desing and Implementation of a 12-Factor Cloud-Native Cafe Order Management System

## Project Summary
This project focuses on developing a cloud-native cafde order management system using ASP.NET Core Web API, Angular, and PostgreSQL. 
The system will support table management, product listing, order item tracking. It will follow 12-factor app principles sucs as environment-based
configuration, stateless services, externalized database usage, containerized deployment, and cloud hosting. The application will be deployed on AWS and
integrated with a CI/CD pipeline using GitHub Actions.

## Technology Stack
- ASP.NET Core Web API
- Angular
- PostgreSQL
- Docker
- AWS
- GitHub Actions

## 12-Factor App Goals 
- Config via environment variables
- Stateless backend services
- Backing services as attached resources
- Dev/prod parity with containers
- Logs to stdout
- Automated build and deployment
